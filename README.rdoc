== Blog

A simple blog app that gives users the opportunity to share their deepest thoughts and comment on other user's posts. Created using Ruby on Rails.

To run this app:
---
  1. clone this repo
  2. set up the database: `rake db:create` then `rake db:migrate`
  3. start the rails server `rails server`
  4. open localhost:3000 in your favorite browser

Future features:
---
* user creation, authentication
* user profile pages
* user avatars
